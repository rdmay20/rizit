rizit

Rizit quickly checks if a webpage has been archived and automatically loads the saved version in a new tab.

How to Get Started

  1. Open Chrome browser on your Android device
  2. Tap the menu (⋮) and select "Install app" or "Add to home screen"
  3. Once installed, you can start using Rizit!

Using Rizit

  1. Browse to any webpage using your preferred browser
  2. Tap the Share button
  3. Select "Rizit" from the share menu
  4. The archived version will load automatically

Desktop Users: Rizit works best on mobile devices, but you can still use it on your desktop browser.

To use Rizit on your desktop:

  1. Copy the URL of the webpage you want to archive.
  2. Paste the URL into the input field below.
  3. Click the "Submit" button to load the archived version.
